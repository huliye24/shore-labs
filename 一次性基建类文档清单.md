# **一次性基建类文档清单**  
（按板块分类，总计约45份核心文件）

### **一、产品与品牌（5份）**
| 文档名称          | 核心内容                                 | 备注           |
| ----------------- | ---------------------------------------- | -------------- |
| 1. 立项申请书     | 市场背景、竞品分析、用户画像、核心价值   | 明确为什么做   |
| 2. 品牌说明书     | 产品名称寓意、视觉风格、Slogan、品牌故事 | 统一内外认知   |
| 3. 产品信息结构图 | App所有页面层级关系图                    | 指导设计和开发 |
| 4. 用户协议       | 平台与用户的权利义务、免责条款           | 法律必备       |
| 5. 隐私政策       | 数据收集、使用、存储、共享规则           | 法律必备       |

---

## **补充说明**
1. **这些文档并非全部需要“从零创作”**：很多模板可以用AI快速生成初稿（如员工手册、合同台账模板），人工根据实际情况调整即可。
2. **部分文档可能需要外部支持**：用户协议、隐私政策、投资相关文件建议由律师审核；商标注册等需要专业代理。
3. **后续会自然衍生出动态文档**：基建完成后，日常运营中会产生大量动态文档（日报、周报、日志等），但这些可以基于已建好的模板和流程自动运转。
