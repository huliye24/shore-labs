# Good First Issue

### 🚀 开发者任务 (Good First Issue)

**任务标题：** 【UI 增强】为“芦花荡” (Reed Pond) 留言卡片添加悬停动效与玻璃材质优化 🌾

**任务描述：** 目前的“芦花荡”匿名广场中，留言卡片的交互感较为单一。为了更贴合“岸” 2.0 的**梦幻、减压**风格，我们需要优化卡片在鼠标悬停时的视觉表现。

**你需要做什么？**

1. **材质升级：** 调整卡片的 `backdrop-filter: blur()` 参数，让毛玻璃效果在奶油色背景下更通透。
2. **动效添加：** 增加一个轻微的浮动效果（微调 `transform: translateY(-5px)`）和柔和的阴影（`box-shadow`）过渡。
3. **响应式微调：** 确保在手机端浏览时，卡片不会因为模糊度过高而导致滑动卡顿。

**相关代码路径：** `src/frontend/components/MessageCard.py` (Streamlit 组件) 或 `static/css/glassmorphism.css`

**预期效果：** 让用户在浏览留言时，感觉到每一张卡片都像是一片轻盈的、漂浮在水面上的羽毛。

**提交要求：**

- 代码需保持模块化。
- 提交 PR 时请附上一张修改前后的对比 GIF 或截图。
